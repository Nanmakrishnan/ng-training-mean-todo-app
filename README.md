MEAN-TODO-Application

A todo-list app project built using MEAN stack

---------Project setup---------

-> Commands to set client side
 1.npm install
 2.npm install -g @angular/cli
 3.ng serve

 -> Commands to set server-side
  1.npm init -y
  2.npm i
  3.npm install -g nodemon  // if nodemon is not installed
  4.nodemon server.js
